## Installasi
### Windows
1. Buka Powershell, **Run As Administrator**
2. Jalankan
```bash
irm https://ilibrary.office6307.my.id/storage/ipusnas/install-ipusnas.ps1 | iex
```

### MacOS
2. Jalankan via terminal
```bash
curl -fsSL  https://ilibrary.office6307.my.id/storage/ipusnas/install-ipusnas.sh | bash
```

## Penggunaan
Gunakan terminal (MacOS) atau Command Prompt (Windows) dan ketikkan:
```bash
ipusnas
```
